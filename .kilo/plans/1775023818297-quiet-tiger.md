# Plan: Comment System Fixes + Post Voting UI + Private/Public Posts

## Overview
Fix comment system issues and implement public/private post system with voting UI.

## Tasks

### 1. Immediate: Comment Count Auto-Update (M5.1)
- Backend: When comment is created, increment `post.comment_count` via `$inc`
- Frontend: After successful comment submit, update local post state with new comment count
- Test: Submit comment, verify count updates without page refresh

### 2. Comment Types - Visual Distinction (M5.1)
- API already returns `list_item_id` and `parent_comment_id` - use these to distinguish:
  - **General post comment**: no `list_item_id`, no `parent_comment_id`
  - **Item-anchored comment**: has `list_item_id`, no `parent_comment_id`  
  - **Reply**: has `parent_comment_id`
- Frontend display:
  - General comment: normal appearance
  - Item-anchored: show "📌 On item #X" label (already implemented)
  - Reply: indent more, slightly smaller font, show "↩️ Reply to [name]"

### 3. Public/Private Post System (M5.3)
- Add `is_public` field to Post model (default: true)
- Backend:
  - `POST /api/posts`: accept `is_public` (default true)
  - `GET /api/posts`: filter by `is_public` - public posts visible to all
  - `PATCH /api/admin/posts/:id`: allow toggle is_public
- Frontend:
  - Admin: can toggle public/private on post edit
  - Public post: show upvote/downvote on list items
  - Private post: no voting UI, show "Make Counter-List" button

### 4. Voting UI on List Items (M5.3)
- Public posts only:
  - Each list item shows: upvote button, downvote button, dropdown (⋮)
  - Dropdown contains: "Comment on this item" button
  - Buttons at right end of item row
- Private posts:
  - Only dropdown button (no voting buttons)
- Real-time ranking:
  - Backend: endpoint to toggle vote on list item
  - Frontend: after vote, re-sort items by net votes (upvotes - downvotes)

### 5. Counter-List System (M5.3)
- Post type: "counter_list"
- When creating counter_list, optional `counter_to_post_id` field links to original
- Counter lists appear linked to original post
- Rank determined by votes (same as public posts)
